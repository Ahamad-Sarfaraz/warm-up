# Java Practice Projects

This repository contains various Java practice projects, design patterns, and coding exercises.

## Project Structure

- `src/LLD/DesignPatterns/` - Implementation of various design patterns
  - `creational/` - Creational design patterns (Factory, Singleton, Prototype, Abstract Factory)
  - `solidPrinciple/` - SOLID principles implementation
- `src/Assesments/` - Coding assessments and challenges
- `src/Basics/` - Core Java concepts and examples
- `src/MultiThreading/` - Multi-threading examples

## Getting Started

1. Clone the repository
2. Open in your preferred Java IDE
3. Run individual files as needed

## Prerequisites
- Java JDK 8 or higher
- Maven (if applicable)
